Addendum

With my JavaScript, I wanted to stay simple and just implement something that would do some math calculations whenever the user clicks a button. Since we had to solve a problem, I decided to go for an algorithm that could help us find prime numbers. It was pretty straightforward. With an if statement, I was able to eliminate integers that are less or equal to one. Then, I eliminated all numbers that are divisible by any numbers other than one and itself. From there, everything was easy: I made a function that my 'verify' button is gonna call whenever clicked. The goal of that function is to read what is inputted in the text box, to make sure the text input is an integer, to check if the inputted numbers are prime, and to display a text for the user.
